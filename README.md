Playfair Cipher
This module implements the Playfair Cipher, a classical encryption/decryption technique that operates on pairs of letters (digraphs) instead of single characters.

How It Works-
A 5×5 matrix is constructed using a keyword (I/J combined).
Plaintext is split into letter pairs.
Encryption rules:
Same row → shift right
Same column → shift down
Rectangle → swap columns
Ciphertext is generated by applying these rules to each pair.

Features-
Dynamic key-based matrix generation
Handles repeated letters using padding (X)
Step-by-step transformation logic
Clean UI-based interaction
Shows the working behind encryption and decryption

Purpose-
Demonstrates how digraph substitution improves security over simple monoalphabetic ciphers by reducing frequency analysis effectiveness.
